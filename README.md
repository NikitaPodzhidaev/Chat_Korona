### Задача по реализации чата с UI-интерфейсом.

#### Реализовано:
#### - Клиент и сервер, которые не зависят друг от друга, при этом запускаются независимо.
#### - Клиент и сервер собираются в разные jar-архивы.
#### - Работа с сетью через чистые сокеты.
#### - Клиент устанавливает соединение с сервером по указанному адресу; пользователю предлагается ввести имя; имеет при себе графический интерфейс (панели вывода, ввода сообщений, список участников чата, кнопки "отправить")
#### - Сервер ждет подключения клиентов на заданном порту(из конфига); рассылает сообщения о подключении различных пользователей.
#### Использовано: Java, Swing, Gradle, SOLID.
